# 📝 To-Do List Web App

## 📌 Project Description

This project is a **To-Do List Web Application** developed as part of my **Web Development Internship at Synent Technologies**. The application allows users to manage their daily tasks efficiently by adding, deleting, and marking tasks as completed.

## 🎯 Purpose of the Project

The purpose of this project is to understand how to build an interactive web application using HTML, CSS, and JavaScript, and to implement basic task management functionality.

## 🚀 Key Features

* Add new tasks
* Delete tasks
* Mark tasks as completed
* Interactive user interface
* Tasks persist using local storage *(if you used it, otherwise remove this line)*

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript

## 📂 Project Structure

* **index.html** → Contains the structure of the application
* **style.css** → Handles styling and layout design
* **script.js** → Contains all the JavaScript logic and functionality

## 📑 Functionalities Included

* Adding tasks dynamically
* Removing tasks from the list
* Marking tasks as completed
* Updating UI based on user actions

## 📚 What I Learned

* Working with JavaScript for DOM manipulation
* Handling user input and events
* Structuring a complete web application
* Managing separate files for HTML, CSS, and JavaScript

## 📌 Conclusion

This project enhanced my understanding of front-end development by building a fully functional and interactive To-Do application using HTML, CSS, and JavaScript.
